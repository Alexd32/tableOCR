Расширение для браузера Chrome

Функционал:
при клике на иконку расширения появляется всплывающее окно рядом с иконкой расширения (справа вверху окна браузера)
В окне заколовок - Table OCR
ниже заколовка большая кнопка с надписью - start
ниже кнопки страт выделенная пунктиром область, которая слудить для перетаскивания файлов. (принимает картинки и pdf файлы только) При клике на область перетаскивания открывается системное окно выбора файла. 
при клике на кнопку "старт" область окна браузера затемняется
курсор мышки видоизменяется на крестик
реализуется функционал выбора области экрана, выбор производится с зажатой левой кнопкой мыши, формируется прямоугольник, 
внутри прямоугольника область осветляется. Выбор заканчивается когда левая кнопка мыши отпущена.
Когда выбор области экрана закончен, то в окне расширения появляется миниатюра выбранной области в виде картинки (отмасштабировання по размеру окна, чтобы не нарушать верстку окна расширения)

При после загрузки файла через выбор файла появляется миниатюра картинки, но если загружен файл pdf то появлеятся не картинка а иконка файла pdf. 
Одновременно можно загрузить только один файл.
